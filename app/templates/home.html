Welcome to the Flask App Homepage!

This application allows you to manage a list of users using a simple and user-friendly interface.

  • Add User: Fill out the form to add a new user with name, email, phone, address, and password.
  • View Users: See all registered users in a styled table format.
  • Edit User: Update existing user information by clicking the edit button.
  • Delete User: Remove users from the database with a single click.

The app uses Flask, SQLAlchemy, HTML templates, CSS styling, and includes Pytest tests and Pylint code quality checks.
